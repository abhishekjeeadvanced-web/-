# Mission "आत्मसम्मान" 🎯
**SSC CGL Ultimate 22-Month Tracker & Mock Analyzer (Progressive Web App)**

A completely offline-first, client-side web application built for SSC CGL aspirants to track their 22-month preparation journey, analyze mock tests, and predict percentiles. 

Designed with a modern **Black & Blue Glassmorphism** theme.

## 🌟 Features
- **Full Syllabus Tracking:** 22 Months structured schedule (Phase 1, 2, and 3).
- **Advanced Mock Analytics:** Log scores, track platforms, and auto-calculate percentiles.
- **Trend Charts:** Weekly, Bi-weekly, and Monthly aggregation charts with sleek gradient lines.
- **100% Secure:** Data is stored locally in your browser (LocalStorage).
- **PWA Ready:** Install it directly to your Mobile/Desktop home screen like a native app.
- **Backup System:** Export and Import your progress as a `.json` file anytime.

## 🚀 How to Deploy on GitHub Pages
1. Create a new repository on GitHub (e.g., `aatmasamman-tracker`).
2. Upload these 4 files into the repository:
   - `index.html`
   - `manifest.json`
   - `sw.js`
   - `logo.svg`
3. Go to your repository **Settings** > **Pages** (on the left sidebar).
4. Under "Build and deployment", set the source to **Deploy from a branch**.
5. Select the `main` branch and `/root` folder, then click **Save**.
6. Wait 1-2 minutes. GitHub will provide you a live link (e.g., `https://yourusername.github.io/aatmasamman-tracker/`).

## 📱 How to Install on Phone/PC
1. Open the live GitHub Pages link in Chrome, Safari, or Edge.
2. A prompt will appear at the bottom: **"Add to Home Screen"** or **"Install App"**.
3. Click it. The app will be installed on your device with the "Mission आत्मसम्मान" logo!
